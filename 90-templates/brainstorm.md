# шаблон: brainstorm

используй когда нужно сгенерировать идеи.

---

## промпт

```
контекст:
[опиши ситуацию — кто ты, что делаешь, какая проблема]

задача:
сгенерируй [N] идей для [чего именно].

требования:
- включи очевидные и неочевидные варианты
- добавь 2-3 "странных" идеи, которые на первый взгляд не подходят
- для каждой идеи: название + 1 предложение почему может сработать

ограничения:
[бюджет / время / ресурсы / что нельзя]

формат:
пронумерованный список
```

---

## пример

```
контекст:
я запускаю курс по AI для не-программистов. целевая аудитория — маркетологи и предприниматели 30-45 лет.

задача:
сгенерируй 10 идей как привлечь первых 50 участников.

требования:
- включи платные и бесплатные каналы
- добавь 2-3 странные идеи
- для каждой: название + почему может сработать

ограничения:
- бюджет $500
- только я, без команды
- 2 недели до запуска

формат:
пронумерованный список
```

---

## после brainstorm

1. выбери 3-5 идей которые резонируют
2. для каждой спроси: "как это сделать за [время]?"
3. выбери одну и начни
