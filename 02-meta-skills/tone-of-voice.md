# tone of voice — как говорить с AI

как ты говоришь с AI влияет на качество ответов.

---

## базовый принцип

AI отражает твой стиль.

- пишешь формально → получаешь формально
- пишешь как человек → получаешь как человек
- пишешь размыто → получаешь воду

---

## что работает

**конкретика:**
```
плохо: "напиши текст про продукт"
хорошо: "напиши пост для телеграма, 200 слов, про то как наш курс помогает не-программистам делать сайты. тон — дружелюбный, без буллшита"
```

**контекст:**
```
плохо: "как мне быть?"
хорошо: "я руковожу командой из 5 человек, один сотрудник постоянно срывает дедлайны. уже говорил с ним дважды. что ещё попробовать?"
```

**роль:**
```
плохо: "дай совет"
хорошо: "представь что ты опытный продакт-менеджер. посмотри на мой план и скажи где слабые места"
```

---

## настройка тона

**для прямоты:**
```
отвечай коротко и по делу. без вступлений и заключений. если я спрашиваю да/нет — отвечай да или нет.
```

**для глубины:**
```
не давай первый очевидный ответ. подумай о неочевидных углах. спроси уточняющие вопросы если надо.
```

**для честности:**
```
если моя идея слабая — скажи прямо. не поддакивай. критикуй конструктивно.
```

**для креатива:**
```
дай 10 вариантов, включая странные и рискованные. не фильтруй заранее.
```

---

## антипаттерны

**слишком вежливо:**
```
"не мог бы ты, пожалуйста, если тебе не сложно..."
→ просто скажи что надо
```

**слишком абстрактно:**
```
"помоги с маркетингом"
→ конкретизируй: какой канал, какая цель, какой бюджет
```

**ожидание магии:**
```
"сделай чтобы было хорошо"
→ опиши что значит "хорошо" для тебя
```

## custom instructions

можно один раз настроить и забыть:

```
стиль общения:
- всё строчными буквами
- короткие предложения
- без воды и мотивационных вставок
- если что-то неясно — спрашивай, не додумывай
- критикуй идеи честно, не поддакивай
```
