# день 5: превращаем контент в лендинг

сегодня берём любой текст — транскрипт звонка, подкаст, статью — и превращаем в красивую интерактивную страницу.

---

## зачем это

у тебя наверняка есть:
- транскрипт созвона который хочется сохранить
- подкаст с крутыми мыслями
- статья которую хочется оформить
- заметки которые жалко терять

обычно это лежит в папке и никто не открывает. а что если превратить это в красивую страницу которую приятно смотреть и которой можно поделиться?

сегодня делаем это за один разговор с cursor.

---

## что делаем

### 1. выбери контент

любой текстовый материал:
- экспорт чата из телеграма
- транскрипт zoom-звонка
- расшифровка подкаста
- статья или пост
- твои заметки

закинь файл в workspace.

### 2. попроси cursor сделать лендинг

открой чат (cmd+L) и напиши что-то типа:

```
вот транскрипт звонка [имя файла]

сделай из него красивый HTML-лендинг:
- вытащи ключевые идеи и цитаты
- оформи с блоками, секциями, карточками
- сделай дизайн минималистичный и стильный
```

cursor создаст HTML-файл. открой его в браузере (правый клик → Open with Live Server или просто открой файл).

### 3. поиграй с дизайном

не нравится — меняй:

```
хочу светлую тему вместо тёмной
```

```
добавь больше воздуха между секциями
```

```
сделай заголовок более цепляющим
```

```
добавь иконки к пунктам
```

### 4. добавь интерактивность

тут начинается магия. попробуй:

```
добавь переключатель темы светлая/тёмная
```

```
добавь прогресс-бар чтения сверху страницы
```

```
сделай так чтобы при клике на заголовок "мы супергерои" вылетало конфетти
```

```
добавь анимацию появления секций при скролле
```

```
сделай карточки кликабельными — чтобы разворачивались
```

### 5. усложняй

когда базовое работает — пробуй сложнее:

```
добавь навигацию по секциям
```

```
сделай фильтр по тегам
```

```
добавь поиск по странице
```

```
сделай чтобы цитаты можно было копировать в один клик
```

---

## примеры что можно превратить в лендинг

| контент | что получится |
|---------|---------------|
| транскрипт Q&A | страница с вопросами-ответами |
| подкаст | ключевые идеи с таймкодами |
| рабочий созвон | summary с решениями и задачами |
| книга/статья | интерактивный конспект |
| свои заметки | красивый knowledge base |

---

## идеи интерактивности

от простого к сложному:

1. **переключатель темы** — светлая/тёмная
2. **прогресс-бар** — показывает сколько прочитал
3. **hover-эффекты** — карточки подсвечиваются
4. **конфетти/анимации** — при клике на что-то
5. **accordion** — секции раскрываются по клику
6. **навигация** — меню с якорями на секции
7. **фильтры** — показать только цитаты / только идеи
8. **поиск** — найти по странице
9. **copy-to-clipboard** — копировать цитаты
10. **share-кнопки** — поделиться секцией
