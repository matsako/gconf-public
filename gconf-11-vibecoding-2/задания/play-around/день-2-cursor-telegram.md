# день 2: cursor читает твои чаты

сегодня cursor начнёт видеть твои переписки. без сложных интеграций — просто скачиваем чат и играемся.

---

## зачем это

телеграм = огромное количество информации которая обычно теряется в потоке. 

представь: ты ищешь ту самую ссылку которую кто-то кидал месяц назад. или хочешь вспомнить о чём договаривались. или понять о чём вообще был разговор в каком-то чате.

сегодня cursor станет твоим поисковиком по чатам.

---

## что делаем

### 1. выбираем чат для эксперимента

выбери что-то интересное:
- чат потока gconf
- рабочий чат где много обсуждений
- личную переписку с кем-то важным
- канал который давно хотел разобрать

### 2. экспортируем чат

**в Telegram Desktop:**
1. открой чат
2. три точки → Export chat history
3. выбери JSON формат (он лучше для анализа)
4. убери галочки с медиа если не нужны
5. экспортируй

получишь папку с файлом `result.json`

### 3. открываем в cursor

1. скопируй папку с экспортом в свой workspace
2. или просто: File → Open Folder → выбери папку с экспортом

### 4. играемся

теперь cursor видит весь чат. спрашивай:

```
о чём вообще этот чат? дай общую картину
```

```
какие темы обсуждались чаще всего?
```

```
найди все ссылки которые кидали
```

```
что писал [имя] про [тема]?
```

```
были ли какие-то договорённости или решения?
```

```
найди самые смешные сообщения
```

```
о чём спорили?
```

---

## идеи что ещё попробовать

- экспортни переписку с близким человеком → попроси cursor найти паттерны в общении
- экспортни рабочий чат → вытащи все задачи и договорённости
- экспортни канал эксперта → сделай выжимку ключевых идей

---

## что получишь

- понимание как cursor работает с большими текстами
- опыт "разговора" с данными
- возможно — найдёшь что-то что давно искал
