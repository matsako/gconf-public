# день 3: cursor + github

сегодня подключаем github и получаем доступ к контексту gconf.

---

## что делаем

### 1. клонируем репозиторий gconf

```bash
git clone https://github.com/matsako/gconf-public.git
```

### 2. открываем в cursor

File → Open Folder → выбираем папку gconf-public

### 3. исследуем через cursor

теперь cursor видит всю базу знаний gconf. спрашивай:

```
что такое framework в gconf?
```
```
как настроить GPT-коуча?
```
```
какие есть use cases для cursor?
```
```
найди примеры cursor rules
```

### 4. пробуем git через cursor

```
какие файлы я изменил?
```
```
сделай коммит с сообщением "мои заметки"
```

---

## зачем github

1. **"кнопка сохранения"** — каждый коммит = точка к которой можно вернуться
2. **синхронизация** — доступ с любого устройства
3. **контекст** — cursor видит всё и может отвечать на вопросы
