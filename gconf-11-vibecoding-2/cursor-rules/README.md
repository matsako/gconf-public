# библиотека cursor rules

готовые правила для cursor от протагонистов и команды.

---

## что такое cursor rules

cursor rules — это инструкции которые cursor применяет ко всем ответам.

можно настроить:
- стиль общения
- формат ответов
- специфику под задачу

---

## как добавить

1. создай файл `.cursorrules` в корне проекта
2. или Settings → Rules for AI

---

## библиотека

```
cursor-rules/
├── general/              # базовые правила
├── тоня-жукова/          # от Тони — здоровье, подкасты, контент
├── [другие протагонисты]/
└── примеры-по-задачам/   # для кода, текстов, анализа
```

---

## как использовать

1. найди правила под свою задачу
2. скопируй в свой `.cursorrules`
3. адаптируй под себя
4. тестируй и корректируй
