# community

сюда участники потока добавляют свои находки.

---

## что можно добавить

- **кейсы** — "что, так можно было?" истории успеха и фейлов
- **cursor rules** — правила которые работают для тебя
- **промпты** — формулировки которые дают результат
- **заметки** — инсайты, выводы, вопросы

---

## как добавить

1. создай файл с понятным названием: `username-тема.md`
2. напиши что хочешь (или попроси cursor помочь)
3. закоммить и запушь

---

## формат не важен

пиши как удобно. главное — чтобы было понятно другим.
