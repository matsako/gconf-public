# gconf public

база знаний для работы с AI через cursor.

---

## как использовать

1. **склонируй репо** — `git clone https://github.com/matsako/gconf-public.git`
2. **открой в cursor** — File → Open Folder → выбери папку
3. **начни разговор** — открой чат (cmd+L) и спроси что угодно про содержимое

cursor видит все файлы в папке и может отвечать на вопросы, искать связи, помогать применять.
