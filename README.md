# gconf public

база знаний для работы с AI через cursor.

---

## как подключиться (для новичков)

чтобы получать обновления когда мы добавляем новые материалы — нужно "подключиться" к репозиторию, а не просто скачать. для этого используем **GitHub Desktop** — приложение с кнопками, без терминала.

### шаг 1: установи GitHub Desktop

1. скачай: https://desktop.github.com
2. установи (просто открой скачанный файл)
3. войди в свой GitHub аккаунт (если нет — создай на github.com)

### шаг 2: склонируй репозиторий

1. открой GitHub Desktop
2. нажми **File** → **Clone Repository**
3. выбери вкладку **URL**
4. вставь: `https://github.com/matsako/gconf-public`
5. выбери куда сохранить (например, рабочий стол)
6. нажми **Clone**

### шаг 3: открой в cursor

1. в GitHub Desktop нажми **Repository** → **Open in Terminal** → скопируй путь к папке
2. или просто запомни куда склонировал
3. открой cursor
4. нажми **File** → **Open Folder**
5. выбери папку `gconf-public`

### шаг 4: начни работать

1. нажми **cmd+L** (или **ctrl+L** на windows) — откроется чат
2. спроси что угодно:
   - "что тут есть?"
   - "с чего мне начать?"
   - "расскажи про framework"

## как получать обновления

когда мы добавляем новые материалы — ты получишь сообщение "обновила репо".

**что делать:**

1. открой **GitHub Desktop**
2. выбери репозиторий `gconf-public` (слева)
3. нажми кнопку **Fetch origin** (наверху)
4. если есть обновления — нажми **Pull origin**

всё. новые файлы появятся в твоей папке.
