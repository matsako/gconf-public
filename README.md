# gconf public

база знаний для работы с AI через cursor.

---

## как скачать и открыть (для новичков)

### шаг 1: скачай репозиторий

1. нажми зелёную кнопку **Code** (наверху этой страницы)
2. выбери **Download ZIP**
3. распакуй скачанный архив в любое удобное место (например, на рабочий стол)

### шаг 2: открой в cursor

1. открой cursor
2. нажми **File** → **Open Folder**
3. выбери папку `gconf-public` (ту, которую распаковал)
4. нажми **Open**

### шаг 3: начни разговор

1. нажми **cmd+L** (или **ctrl+L** на windows) — откроется чат
2. спроси что угодно, например:
   - "что тут есть?"
   - "с чего мне начать?"
   - "расскажи про framework"

cursor видит все файлы и может отвечать на вопросы, объяснять, помогать применять.

## с чего начать

1. открой `01-start-here/what-is-vibe-coding.md`
2. попробуй `01-start-here/quick-wins.md`
3. выбери один use case из `04-use-cases/`

или просто спроси cursor: "с чего мне начать?"

## как получать обновления

мы добавляем новые материалы по ходу потока.

**простой способ:** скачай ZIP заново (шаг 1) и замени папку.

**продвинутый способ (если настроил git):**
```bash
git pull
```
