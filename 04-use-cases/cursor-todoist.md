# cursor + todoist

как управлять задачами через cursor.

---

## зачем это

- не переключаться между окнами
- создавать задачи голосом/текстом
- AI сам формулирует и структурирует
- всё в одном месте

---

## как настроить

через MCP (Model Context Protocol) cursor получает доступ к твоему todoist.

**шаги:**

1. получи API token в todoist (Settings → Integrations → Developer)
2. настрой MCP в cursor

## что можно делать

**создавать задачи:**
```
создай задачу "позвонить клиенту" на завтра в 10:00
```

**смотреть задачи:**
```
что у меня на сегодня?
какие задачи просрочены?
```

**управлять:**
```
перенеси задачу X на пятницу
отметь задачу Y выполненной
```

**умные запросы:**
```
какие задачи по проекту "gconf"?
что я откладываю уже неделю?
```

---

## продвинутые кейсы

**AI создаёт задачи из контекста:**
```
прочитай заметки с созвона и создай задачи для каждого action item
```

**приоритизация:**
```
посмотри мои задачи на неделю и предложи как их приоритизировать
```

**рефлексия:**
```
сколько задач я закрыл за прошлую неделю? какие паттерны видишь?
```
