# cursor + youtube

как вытаскивать контекст из видео и подкастов.

---

## зачем это

- подкаст 2 часа, а времени нет
- хочется выжимку под свой запрос
- нужно найти конкретный момент
- хочется сохранить инсайты

---

## как это работает

1. получаешь транскрипт видео
2. кладёшь в cursor
3. спрашиваешь что нужно

---

## способ 1: через сервисы транскрипции

**youtube transcript:**
- много расширений для chrome
- копируешь транскрипт → в файл → в cursor

**tactiq / otter / fireflies:**
- автоматически транскрибируют
- экспорт в текст

---

## способ 2: через MCP (автоматически)

настраиваешь MCP для youtube → cursor сам достаёт транскрипт по ссылке.

## что спрашивать

**выжимка:**
```
дай главные идеи из этого подкаста в 5 пунктах
```

**под свой запрос:**
```
я маркетолог. что из этого видео применимо к моей работе?
```

**конкретный вопрос:**
```
что они говорят про pricing? найди этот момент
```

**action items:**
```
какие конкретные действия рекомендуют? составь список
```

---

## пример

**видео:** 2-часовой подкаст про здоровье

**запрос:**
```
я хочу улучшить сон. что из этого подкаста мне применить?
дай 3 конкретных действия с объяснением почему.
```

**результат:** персонализированная выжимка за 2 минуты вместо 2 часов просмотра.

---

## лайфхаки

- сохраняй полезные транскрипты в отдельную папку
- добавляй свои заметки в тот же файл
- потом можешь спрашивать cursor "что я смотрел про X?"

---

*контент бесконечен. cursor помогает брать только нужное.*
