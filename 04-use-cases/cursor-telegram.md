# cursor + telegram

как получить доступ к своему телеграму через cursor.

---

## зачем это

- искать по всем чатам одним запросом
- анализировать обсуждения
- вытаскивать нужное без бесконечного скролла
- связывать с другими данными

---

## два способа

### способ 1: экспорт чата (простой)

1. открой телеграм desktop
2. выбери чат → три точки → Export Chat History
3. выбери формат JSON
4. положи папку с экспортом в свой cursor workspace
5. спрашивай cursor про содержимое

**пример:**
```
проанализируй этот чат и скажи:
- какие темы обсуждались чаще всего?
- кто был самым активным?
- какие решения были приняты?
```

**плюсы:** просто, не нужна настройка
**минусы:** статичный снимок, надо экспортировать заново

---

### способ 2: telethon + MCP (продвинутый)

telethon — библиотека для работы с Telegram API. через MCP cursor получает live доступ к твоему аккаунту.

**что даёт:**
- актуальные данные (не снимок)
- доступ ко всем чатам
- можно отправлять сообщения через cursor

**как настроить:**

1. получи API credentials на my.telegram.org
2. установи telethon: `pip install telethon`
3. настрой MCP в cursor

---

## кейсы использования

**для работы:**
- "собери все задачи которые мне ставили в рабочем чате за неделю"
- "найди где мы обсуждали бюджет на Q1"

**для обучения:**
- "какие инсайты были в чате потока?"
- "кто делился похожим кейсом?"

**для жизни:**
- "когда мы договаривались встретиться с X?"
- "о чём мы говорили с Y в прошлом месяце?"

---

## предупреждение

отправка сообщений через telethon — рискованно. телеграм может заблокировать аккаунт за подозрительную активность.

для чтения — безопасно.

---

*телеграм = огромное количество информации. cursor помогает её использовать.*
