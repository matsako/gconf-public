# quick wins — первые простые штуки

попробуй эти вещи чтобы почувствовать что такое vibe coding.

---

## 1. поговори с cursor про себя

открой чат (cmd+L) и скажи:

```
привет! я [кто ты]. расскажи что я могу делать с тобой?
```

просто познакомься. потыкай. спроси глупые вопросы.

**время:** 5 минут

---

## 2. создай файл через агента

напиши в чат:

```
создай файл my-notes.md и напиши туда 3 идеи проектов которые я мог бы сделать с AI
```

посмотри как cursor создаёт файл и наполняет его.

**время:** 2 минуты

---

## 3. попроси объяснить код

найди любой файл с кодом (или создай) и спроси:

```
объясни что делает этот файл простыми словами
```

cursor разберёт и объяснит.

**время:** 3 минуты

---

## 4. сделай простую автоматизацию

напиши:

```
напиши python скрипт который берёт список задач из файла tasks.txt и сортирует их по приоритету
```

даже если не знаешь python — получишь работающий скрипт.

**время:** 10 минут

---

## 5. подключи свой контекст

скачай экспорт любого своего телеграм-чата (Settings → Export) и положи в папку.

потом спроси:

```
проанализируй этот чат и скажи какие темы там обсуждались чаще всего
```

cursor прочитает и ответит.

**время:** 15 минут
