# как использовать этот репозиторий

---

## шаг 1: склонируй к себе

```bash
git clone https://github.com/matsako/gconf-public.git
```

или скачай как ZIP и распакуй.

---

## шаг 2: открой в cursor

1. открой cursor
2. File → Open Folder
3. выбери папку gconf-public

теперь cursor видит всё содержимое.

---

## шаг 3: начни разговор

открой чат с агентом (cmd+L или ctrl+L) и спроси что угодно:

- "что такое метанавыки?"
- "как подключить телеграм к cursor?"
- "покажи пример coach prompt"
- "что мне попробовать первым делом?"

cursor найдёт релевантные файлы и ответит на основе контекста.

---

## как это работает

cursor читает все файлы в папке и использует их как контекст для ответов.

это значит:
- ты можешь спрашивать про любую тему из репо
- cursor будет давать ответы на основе наших материалов
- чем конкретнее вопрос — тем лучше ответ
